# پروژه شبیه‌سازی Batmobile در Godot 4

## معرفی پروژه
این پروژه یک پیاده‌سازی سه‌بعدی از خودروی بتمن (Batmobile) با استفاده از موتور بازی‌سازی **Godot Engine 4** است. هدف اصلی این پروژه، پیاده‌سازی فیزیک حرکت آرکید خودرو، مدیریت حالت‌های مختلف (State Machine) و تعامل با محیط از طریق نورپردازی و صداگذاری پویا است.

در این شبیه‌سازی، بازیکن کنترل کامل خودرو را در دست دارد و می‌تواند بین حالت‌های مختلف (عادی، مخفی‌کاری و هشدار) سوییچ کند و از ابزارهایی مانند Bat-Signal استفاده نماید.

## کنترل‌ها (Controls)

لیست کامل کلیدهای تعریف شده برای تعامل با بت‌موبیل:

| کلید (Key) | عملکرد (Action) | توضیحات |
| :--- | :--- | :--- |
| **W / A / S / D** | **حرکت ماشین** | حرکت به جلو، عقب و چرخش به چپ و راست. |
| **Space** | **حالت هشدار (Alert)** | فعال‌سازی آژیر، تغییر رنگ چراغ‌ها به قرمز و افزایش سرعت نهایی. |
| **C** | **حالت مخفی (Stealth)** | کاهش سرعت، تغییر نور به آبی کم‌رنگ و حرکت بی‌صدا. |
| **N** | **حالت عادی (Normal)** | بازگشت به سرعت و نورپردازی استاندارد (سفید). |
| **Shift** | **Boost** | افزایش سرعت لحظه‌ای (در حالت عادی). |

## خلاصه‌ی کار انجام‌شده)

در توسعه این پروژه، موارد فنی زیر پیاده‌سازی و کدنویسی شده‌اند:

1.  **سیستم حرکتی (Physics Based Movement):**
    *   استفاده از `CharacterBody3D` برای مدیریت فیزیک.
    *   پیاده‌سازی شتاب (`acceleration`)، ترمز (`deceleration`) و اصطکاک برای ایجاد حس طبیعی رانندگی.

2.  **ماشین حالت (State Machine):**
    *   استفاده از `Enum` برای مدیریت سه حالت اصلی: **NORMAL**, **STEALTH**, **ALERT**.
    *   تغییر پویای متغیرهایی مثل سرعت (`speed`)، انرژی نور (`light_energy`) و رنگ نور (`light_color`) بر اساس حالت فعلی.

3.  **مدیریت نورپردازی و افکت‌ها:**
    *   اصلاح جهت‌گیری و برد نودهای `SpotLight3D` برای چراغ‌های جلو.

4.  **سیستم صوتی:**
    *   مدیریت پخش صدای آژیر (`SirenSound`) که فقط در حالت **Alert** پخش می‌شود و در سایر حالت‌ها قطع می‌گردد.
    *   رفع باگ‌های مربوط به تکرار (Loop) و قطع شدن ناگهانی صدا.

5.  **عیب‌یابی و بهینه‌سازی:**
    *   حل مشکلات مربوط به `Null Instance` در نودها.
    *   تطبیق سیستم ورودی (`Input Map`) با اسکریپت‌ها.
    مدیریت ورژن و بازیابی فایل‌ها از طریق Git.
